# 项目工作流：万能生活清单助手本地缓存版

## 当前边界

- 不接微信支付
- 不接云后台
- 不使用云函数、云数据库
- 不引入传统服务器
- 不引入前端 npm 依赖
- 用户数据全部保存在微信小程序本地缓存中

## 开发顺序

1. 保证小程序能在微信开发者工具中编译。
2. 优先打磨首页、分类页、清单详情页、我的页。
3. 扩充生活清单模板。
4. 优化本地生成清单、常用清单、今日要做、选择困难症助手。
5. 做上线前测试报告。

## 验收重点

- `app.json` 页面路径正确。
- 所有页面 `.js`、`.wxml`、`.wxss`、`.json` 文件齐全。
- JS 语法通过。
- WXML 标签闭合。
- 本地缓存读写正常。
- 保存图片到相册权限说明清楚。
- 不出现手机号、支付、服务器、远程同步相关能力。
