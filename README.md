# 万能生活清单助手 - 微信小程序本地缓存版

这是一个原生微信小程序项目，定位为轻量的生活清单工具。

## 项目特点

- 不接微信支付
- 不强制获取手机号、头像、昵称
- 不接云后台、不使用云函数、不使用云数据库
- 用户数据全部保存在微信小程序本地缓存中
- 使用原生 WXML、WXSS、JavaScript、JSON
- 不引入前端 npm 依赖

## 已实现功能

- 首页精选模板和常用清单入口
- 分类页按生活场景浏览模板
- 清单详情页支持勾选、添加事项、删除事项、重置
- 生成“我的清单”，保存到本地缓存
- 我的清单中心查看、打开、删除本地清单
- 今日要做聚合进行中的清单和今日完成记录
- 选择困难症助手
- 生成清单图片并保存到相册
- 普通小程序分享入口

## 本地数据说明

用户数据使用 `wx.setStorageSync` 保存在本机缓存中，包括：

- 清单勾选草稿
- 最近生成的清单
- 常用清单
- 今日完成记录
- 选择困难症助手历史

卸载小程序、清理微信缓存或更换设备后，本地数据可能丢失。本版本不提供跨设备同步。

## 运行方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择当前目录。
4. AppID 使用自己的小程序 AppID。
5. 点击编译。

## 上线注意

- 本版本不需要开通云开发。
