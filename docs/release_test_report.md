# 上线前测试报告：本地缓存版

检查日期：2026-05-26

## 结论

当前版本已切换为纯本地缓存版。小程序不依赖云后台，不使用云函数，不使用云数据库，不获取 openid。用户生成的清单、勾选草稿、常用清单、今日完成记录和选择历史均保存在微信小程序本地缓存中。

## 已移除内容

- 云开发初始化逻辑
- 云函数调用封装
- 云端用户数据同步
- 云端模板读取
- 共享清单快照
- 小程序码海报生成
- 管理员中心
- 云函数目录文件
- 云开发部署文档

## 当前主流程

- 首页：展示常用清单、精选模板、查看清单入口、选择困难症助手入口。
- 分类页：只读取本地 `data/templates.js`。
- 清单详情页：支持勾选、添加事项、删除事项、重置、生成我的清单、保存图片。
- 我的页：展示本地生成的清单、常用清单和数据保存状态。
- 今日要做：聚合本地进行中的清单和今日完成记录。
- 选择困难症助手：本地保存历史记录。

## 隐私与权限

- 不获取手机号。
- 不获取头像昵称。
- 不获取 openid。
- 不接微信支付。
- 不使用服务器。
- 保存图片时会申请写入相册权限。
- 不提供意见反馈入口。

## 需要真机验证

1. 首页模板是否正常进入详情页。
2. 详情页勾选、添加、删除、重置是否正常。
3. 点击“生成我的清单”后，我的页是否能看到新清单。
4. 保存清单图片时，相册权限是否正常。
5. 清理微信缓存或卸载后，本地数据丢失是否符合预期。

## 潜在风险

- 本地缓存无法跨设备同步。
- 用户清理缓存后数据会丢失。
- 已生成的本地自定义清单分享给别人时，别人无法打开同一份个人清单，只能分享小程序入口或保存图片。
