# 万能生活清单助手 AI 员工配置

这份文档用于在 Codex 中组织多个 AI 员工分批协作。当前项目是微信小程序“万能生活清单助手”，第一版目标是：个人主体可上线、不接微信支付、不接云后台、优先完成可运行 MVP。

## 团队总原则

1. 先做整体方案，再按阶段实现，不要一上来堆功能。
2. 当前阶段不接微信支付、不做订单、不做交易闭环。
3. 当前阶段不接云后台，除非任务明确要求只输出云开发二期方案。
4. 不做头像、文案生成、AI 生成器类功能。
5. 所有员工输出都必须说明：目标、影响文件、修改内容、验证方式、潜在风险。
6. 任何涉及老人、母婴、就医、装修、法律、金融的内容，只能做生活准备清单，不输出诊断、治疗、投资、法律结论。
