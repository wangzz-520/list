# Codex 可执行任务：本地缓存版

当前项目为“万能生活清单助手”微信小程序本地缓存版。

固定边界：

- 不接微信支付
- 不接云后台
- 不新增云函数
- 不新增云数据库
- 不引入传统服务器
- 不引入前端 npm 依赖
- 用户数据全部保存在本地缓存

## 常用任务

### 任务 1：检查能否编译

检查 `app.json`、`project.config.json`、页面路径、JS require 路径、WXML 绑定变量。发现问题直接修复，并输出验证步骤。

### 任务 2：优化 UI

优化首页、分类页、清单详情页、我的页的布局、按钮、空状态和文字层级。保持主色 `#2f6fed`。

### 任务 3：扩充模板

在 `data/templates.js` 中增加生活场景清单。模板必须包含 `id`、`title`、`category`、`icon`、`heat`、`description`、`groups`。

### 任务 4：增强本地清单管理

用户在“我的”页能查看、打开、删除本地生成的清单。涉及存储结构时必须兼容旧数据。

### 任务 5：上线前测试

输出 `docs/release_test_report.md`，检查页面路径、JS 语法、WXML 标签、本地存储、分享路径、相册权限、隐私边界。
